***Number Guessing Game***

 ##Project Description
 This is a simple ***JAVA-Based Number Game*** where the computer generates a random number between 1 and 100, and the user tries to guess it within a limited number of attempts.

The program provides feedback on each guess (too high, too low, or correct) and allows the user to play multiple rounds. It also tracks the user's score based on performance.

##Features:
- Random number generation (1 to 100)
- User input for guessing the number
- Feedback for each guess:
   - Too high
   - Too low
   - Correct
- Limited number of attempts
- Multiple rounds (play again option)
- Score calculation based on attempts

  ##Technologies Used:
- Java
- Scanner class for input handling
- Random class for number generation
